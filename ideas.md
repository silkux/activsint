# Brainstorming: Visualizador de Eventos de Activismo

## Concepto General
Un visualizador interactivo que mapea eventos de alto impacto social (como la muerte de Kshamenk, la orca en cautiverio) con variables de impacto mediático, ubicación geográfica, y potencial de movilización activista. El sistema notifica a activistas en zonas relevantes para maximizar la precisión y efectividad de las acciones.

## Respuesta 1: Enfoque Activista Radical - Movimiento de Justicia Social

**Probabilidad: 0.08**

<response>
<text>

### Design Movement
Diseño inspirado en el **activismo gráfico de los años 70-80** y **movimientos de justicia social contemporáneos**. Estética de carteles de protesta, tipografía audaz, colores que generan urgencia y contraste visual extremo.

### Core Principles
1. **Urgencia Visual**: Cada elemento comunica importancia inmediata. Los eventos críticos saltan del diseño.
2. **Accesibilidad Radical**: Tipografía grande, contraste extremo, sin elementos decorativos que distraigan.
3. **Narrativa de Poder**: El usuario es el protagonista. Los datos sirven para empoderar acciones concretas.
4. **Transparencia Activista**: Mostrar claramente cómo se calcula el impacto, quién puede actuar, qué resultados se esperan.

### Color Philosophy
- **Paleta primaria**: Negro profundo + Rojo sangre + Blanco puro
- **Acentos**: Amarillo neón para alertas, Verde esperanza para victorias activistas
- **Reasoning**: Los colores evocan urgencia, peligro, y acción. El rojo representa sangre derramada (animales), el amarillo es visibilidad, el verde es esperanza de cambio.

### Layout Paradigm
- **Estructura asimétrica y dinámica**: Bloques de contenido en ángulos, no alineados a grid
- **Mapa como protagonista**: Ocupar 60% del espacio, con eventos como "puntos de acción"
- **Sidebar de impacto**: Columna izquierda con métricas de impacto en tiempo real
- **Flujo de lectura diagonal**: Guiar ojos hacia acciones concretas

### Signature Elements
1. **Iconografía de puño/acción**: Símbolos de resistencia integrados en cada evento
2. **Líneas de conexión dinámicas**: Mostrar relaciones entre eventos, causas y activistas
3. **Badges de urgencia**: Etiquetas que indican "Acción Inmediata", "Cobertura Mediática Alta", etc.

### Interaction Philosophy
- **Click = Acción**: Cada clic debe llevar a una acción concreta (contactar activistas, compartir, firmar)
- **Feedback inmediato**: Animaciones que celebran acciones (confeti, checkmarks)
- **Gamificación activista**: Puntos por participación, logros desbloqueados

### Animation
- **Entrada de eventos**: Explosión/aparición rápida con sonido (opcional)
- **Hover en eventos**: Pulse rojo, información se despliega agresivamente
- **Transiciones**: Rápidas y directas, sin suavidad excesiva
- **Notificaciones**: Slide desde arriba con urgencia

### Typography System
- **Display**: Fuente sans-serif pesada (Bebas Neue, Oswald) para títulos
- **Body**: Fuente sans-serif clara (Roboto, Open Sans) para legibilidad
- **Hierarchy**: Títulos 48px+, body 14px, con mucho contraste
- **Estilo**: Mayúsculas para llamadas a acción, minúsculas para contexto

## Respuesta 2: Enfoque Científico-Ambiental - Visualización de Datos Sofisticada

**Probabilidad: 0.07**

<response>
<text>

### Design Movement
Inspirado en **infografías científicas**, **dashboards de sostenibilidad**, y **visualización de datos de organizaciones ambientales como WWF o Greenpeace**. Elegancia minimalista con énfasis en datos precisos y patrones visuales.

### Core Principles
1. **Precisión Estética**: Cada píxel comunica datos. No hay decoración sin propósito.
2. **Jerarquía de Información**: Los datos más críticos son los más visibles y legibles.
3. **Conexión Emocional a través de Datos**: Los números cuentan historias. Visualizar el impacto de forma que toque emocionalmente.
4. **Confianza y Credibilidad**: Diseño que comunica rigor científico y precisión.

### Color Philosophy
- **Paleta primaria**: Azul océano profundo + Verde bosque + Blanco limpio
- **Acentos**: Naranja para alertas, Púrpura para datos históricos
- **Reasoning**: Azul evoca océano (Kshamenk), verde evoca naturaleza, naranja es precaución científica. La paleta comunica seriedad ambiental.

### Layout Paradigm
- **Grid de 12 columnas**: Estructura ordenada y predecible
- **Mapa central con paneles laterales**: Mapa interactivo en el centro, métricas en paneles deslizables
- **Timeline vertical**: Mostrar evolución temporal de eventos
- **Gráficos integrados**: Pequeños charts (líneas, barras) que muestran tendencias

### Signature Elements
1. **Nodos de red**: Eventos como nodos conectados por líneas de causalidad
2. **Heatmaps**: Mostrar densidad de activismo por región
3. **Iconografía científica**: Símbolos de especies, tipos de eventos, niveles de impacto
4. **Badges de datos**: Etiquetas con números (cobertura mediática, activistas movilizados, etc.)

### Interaction Philosophy
- **Exploración libre**: Usuario descubre patrones por sí mismo
- **Tooltips informativos**: Hover revela datos sin abrumar
- **Filtros avanzados**: Por tipo de evento, región, impacto, fecha
- **Exportación de datos**: Permitir descargar reportes

### Animation
- **Transiciones suaves**: Easing cúbico para cambios de vista
- **Entrada de datos**: Animación de "dibujo" para gráficos (líneas que se trazan)
- **Hover sutil**: Cambio de opacidad, no movimiento agresivo
- **Loading states**: Esqueletos de carga que comunican estructura

### Typography System
- **Display**: Fuente serif elegante (Merriweather, Lora) para títulos principales
- **Body**: Fuente sans-serif limpia (Source Sans Pro, Inter) para datos
- **Monospace**: Para números y métricas (Roboto Mono)
- **Hierarchy**: Títulos 36px, subtítulos 18px, body 14px

</text>

## Respuesta 3: Enfoque Comunitario-Participativo - Red Social de Activismo

**Probabilidad: 0.09**

<response>
<text>

### Design Movement
Inspirado en **diseño de redes sociales comunitarias**, **plataformas colaborativas** (como Wikipedia), y **movimientos de base local**. Calidez, inclusión, y sensación de comunidad. Colores vibrantes pero armoniosos.

### Core Principles
1. **Comunidad Primero**: El diseño celebra a las personas, no solo los datos.
2. **Accesibilidad Inclusiva**: Múltiples formas de interactuar (visual, textual, auditiva).
3. **Empoderamiento Local**: Enfatizar acciones a nivel comunitario, no solo global.
4. **Narrativa Humana**: Historias de activistas reales, no solo números.

### Color Philosophy
- **Paleta primaria**: Turquesa vibrante + Coral cálido + Crema
- **Acentos**: Limón para acciones, Lavanda para reflexión
- **Reasoning**: Turquesa evoca agua (océano, vida), coral es energía humana, crema es calidez. La paleta comunica esperanza y comunidad.

### Layout Paradigm
- **Estructura modular y flexible**: Cards que se reorganizan según interés del usuario
- **Mapa como contexto, no protagonista**: Mapa pequeño en sidebar, eventos como cards principales
- **Feed de actividad**: Timeline de eventos recientes con comentarios/acciones de activistas
- **Perfiles de activistas**: Mostrar quién está actuando, crear conexiones entre personas

### Signature Elements
1. **Avatares de activistas**: Mostrar caras/nombres de personas involucradas
2. **Badges de comunidad**: "Apoyado por X activistas", "Trending en tu región"
3. **Historias de impacto**: Cards que cuentan historias de cambio logrado
4. **Llamadas a acción personalizadas**: "Hay 5 activistas en tu ciudad trabajando en esto"

### Interaction Philosophy
- **Social first**: Compartir, comentar, conectar con otros activistas
- **Recomendaciones personalizadas**: "Si te importa X, te interesará Y"
- **Celebración de acciones**: Mostrar logros, no solo problemas
- **Comunidad como recurso**: Conectar activistas entre sí para colaboración

### Animation
- **Entrada de cards**: Slide suave desde abajo con bounce
- **Hover en perfiles**: Tarjeta se expande mostrando más información
- **Reacciones sociales**: Animaciones de "me gusta" con emojis
- **Notificaciones**: Toast amigable que invita a participar

### Typography System
- **Display**: Fuente redondeada y amigable (Poppins, Quicksand) para títulos
- **Body**: Fuente sans-serif cálida (Nunito, Raleway) para legibilidad
- **Accent**: Fuente script ligera para citas/testimonios
- **Hierarchy**: Títulos 40px, subtítulos 20px, body 15px, con espaciado generoso

## Selección: Enfoque Científico-Ambiental

Después de considerar los tres enfoques, **selecciono el Enfoque Científico-Ambiental** como la dirección principal. Este enfoque es óptimo porque:

1. **Credibilidad**: Los activistas necesitan datos precisos y confiables para tomar decisiones.
2. **Escalabilidad**: La estructura de datos y visualización puede crecer sin perder claridad.
3. **Impacto Emocional + Racional**: Combina números con narrativa visual que toca emocionalmente.
4. **Profesionalismo**: Comunica seriedad y rigor, atrayendo tanto a activistas como a medios.

### Paleta de Colores Final
- **Primario**: `#0F5A7F` (Azul océano)
- **Secundario**: `#2D8659` (Verde bosque)
- **Alerta**: `#E67E22` (Naranja precaución)
- **Fondo**: `#F8FAFB` (Blanco limpio)
- **Texto**: `#1A1A1A` (Negro profundo)

### Tipografía
- **Display**: Merriweather (serif, 36px+)
- **Body**: Inter (sans-serif, 14-16px)
- **Monospace**: Roboto Mono (para métricas)

### Componentes Clave
1. Mapa interactivo con eventos como nodos
2. Paneles laterales con métricas y filtros
3. Timeline vertical de eventos
4. Gráficos de tendencias
5. Sistema de notificaciones
